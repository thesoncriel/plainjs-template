# PlainJS Template
간단한 js 프로젝트를 진행하기 위한 프로젝트 템플릿이다.  

## Getting Started
개발 진행 시 아래 명령어로 수행 한다.
```
npm run dev
```
그러면 기본 웹브라우저에서 localhost:8080 주소로 자동으로 열릴 것이다.

## Features
- webpack-dev-server 를 이용하여 개발 진행 시 실시간으로 개발 변화 내용을 볼 수 있도록 함. (HMR 이용)
